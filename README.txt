-------!!! Инструкция по установке enChat !!!---------------

Для развертывания приложения необходимо наличие 3 компонентов:
 - Java Runtime Enviroment - среда выполнения Java приложения (jre-8uu....)
 - контейнер сервлетов Tomcat (apache-tomcat-8.5.45)
 - сервер базы данных, в данном случае MYSQL или PostgreSQL (установка/настройка баз данных полностью возлагается на Вас)
 Помимо этого необходимо создать бд с именеи "esn". И обеспечить к ней полный доступ пользователя, который будет
 использован для связи приложения с бд. Кодировку для бд установить - utf8.

Ссылки для скачивания продуктов:
https://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html
https://tomcat.apache.org/download-80.cgi
https://downloads.mysql.com/archives/community/
https://www.postgresql.org/download/

- после установки tomcat переместитесь в директорию  ..\Apache Software Foundation\Tomcat 8.5\webapps и очистите её.
- скопируйте файл приложения ROOT.war в директорию \webapps
- измените в файле conf/server.xml настройки соединения

<Connector port="8080" protocol="HTTP/1.1"
               connectionTimeout="20000"
               redirectPort="8443" />

               на

<Connector port="8080" protocol="HTTP/1.1"
              URIEncoding="UTF-8"
              connectionTimeout="20000"
              redirectPort="8443" />

- перезапустите системную службу tomcat

После перезапуска службы ROOT.war будет распакован в директорию ROOT
- откройте файл с настройками приложения в файле ROOT/WEB-INF/classes/properties и отредактируйте необходимые значения.
Это единственный файл, в который можно вносить изменения. Обязательно укажите тип базы данных, логин, пароль.
Директория файлового хранилища и служебных файлов будет создана в ROOT\resources\data\[URL организации]
- после настройки снова перезапустите службу tomcat
Через некоторое время приложение будет доступно на порту 8080 (по умолчанию).
